eflowcalc comes with a calculator that pre-processes the input arrays and calculates the desired streamflow characteristic(s) (SFC(s)).

Streamflow Characteristics

eflowcalc features 156 SFCs categorised into five categories focussing on flow duration, flow magnitude, flow frequency, flow timing, and flow rate of change, respectively. Each category can be subdivided into low flow, average flow, and or high flow events.

  • Characteristics on flow duration
    • Low flow events
    • High flow events
  • Characteristics on flow magnitude
    • Average flow events
    • Low flow events
    • High flow events
  • Characteristics on flow frequency
    • Low flow events
    • High flow events
  • Characteristics on flow rate of change
    • Average flow events
  • Characteristics on flow timing
    • Average flow events
    • Low flow events
    • High flow events
