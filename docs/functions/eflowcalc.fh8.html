eflowcalc.fh8¶

eflowcalc.fh8(flows, datetimes, hydro_years, drainage_area)¶

Frequency of events above Q25.

Note: this streamflow characteristic is equivalent to fh1.

Calculation Details

Identify for each hydrological year the flow events above the 75th percentile of the whole daily flow record. Count the number of these flow events for each hydrological year. Calculate the mean of these number of events.
